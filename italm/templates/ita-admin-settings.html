iTMS Link Generator URL
This is the URL for the iTunes Link maker, you can enter the entire URL provide from your affiliate program here if you wish, alternatively you can enter the standar URL here, and the affiliate extras below.
Partner Link Stuff
Whatever is in this box, will be merged onto the end of the iTMS Generator URL above.
Default Link Image
URL of the image that will be used to link to the store when no text is selected.
Default Media Category
The popup window will default to this media type everytime it loads.
Default Country
The popup window will default to this country everytime it loads.
